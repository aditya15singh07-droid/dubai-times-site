# Vercel Hosting

Use Vercel for temporary hosting while Supabase remains the database layer for the backend engine.

## Recommended Project

```text
dubai-times-site
```

## Build Settings

```text
Framework Preset: Astro
Build Command: npm run build
Output Directory: dist
Install Command: npm install
Production Branch: master
```

## Temporary Domain

Vercel will provide a public domain like:

```text
https://dubai-time.com
```

## Custom Domain

To use a branded domain, first buy one, then add it in:

```text
Vercel Project -> Settings -> Domains
```

Good options:

```text
dubaitimes.news
dubaitimes.media
dubaitimesdaily.com
```

After adding the domain, Vercel will show DNS records to add wherever the domain is registered.
